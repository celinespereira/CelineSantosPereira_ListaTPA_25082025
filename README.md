## Exercícios e Projetos de Programação

Este repositório reúne as atividades desenvolvidas durante as aulas, com o objetivo de praticar lógica de programação e o desenvolvimento de aplicações utilizando C#.

## Conteúdo

- 10 exercícios propostos em sala de aula.
- Jokenpô.
- Jogo da Velha.

## Objetivos

- Praticar lógica de programação.
- Desenvolver soluções para problemas propostos.
- Aprimorar conhecimentos em C#.
- Aplicar conceitos aprendidos em aula por meio de pequenos projetos.

## Tecnologias Utilizadas

- C#
